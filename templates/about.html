About the Azure Load Balancer Test App

This application is designed to test Azure internal load balancer functionality by allowing you to:

  • Test HTTP connectivity to internal IP addresses in your Azure network
  • Verify load balancer rules are working correctly
  • Check direct server-to-server communication
  • Troubleshoot networking issues

How It Works

The application makes HTTP/HTTPS requests to specified IP addresses and ports within your Azure internal network. It displays detailed information about the connection attempt, including:

  • Connection success or failure
  • Response time
  • HTTP status codes
  • Response content (if available)

Usage Scenarios

This tool is particularly useful when:

  • Setting up new load balancer rules
  • Troubleshooting connectivity issues between services
  • Verifying network security group (NSG) configurations
  • Testing server health
